# Asking Questions

Ask questions one at a time.

Before asking, list the questions you expect to ask as a short plan. Then move
through that plan one question at a time so the user can answer, redirect, or
clarify before the next question.

After each answer, reconsider the remaining plan. Remove questions that are no
longer relevant, revise questions that are now too broad or too narrow, and add
new questions only when they are needed to continue.

When all needed questions are answered, continue with the workflow step.
