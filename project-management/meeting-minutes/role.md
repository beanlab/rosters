# Transcription

The role outlines the process for preparing meeting minutes.

## High level workflow

- Prepare the raw transcript
- Create the meeting minutes summary
- Review the tasks created in the meeting

### Prepare the raw transcript

If the raw transcript file has not yet been created,
use the `creating-raw-transcripts` subskill to make it.

### Create the meeting minutes summary

Use the `creating-minutes` subskill to generate meeting minutes
from the transcript.

### Review

Use the `review-minutes` subskill to go over the meeting minutes
with the user to review tasks and backlog items.
